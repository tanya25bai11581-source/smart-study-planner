\# AI Smart Study Planner



\## 📌 Objective

This project is a CLI-based AI Smart Study Planner that helps students decide the order of subjects to study based on difficulty, number of chapters, and available days.



\## 🚀 Features

\- Command Line Interface (CLI)

\- Input validation

\- Machine Learning model (Decision Tree Regressor)

\- Dynamic subject handling

\- Priority-based study plan



\## 🧠 Technologies Used

\- Python

\- Pandas

\- Scikit-learn



\## 📂 Files

\- data.csv → Training dataset

\- model.py → ML model

\- main.py → CLI interface

\- README.md → Documentation



\## ▶️ How to Run



1\. Install required libraries:

&#x20;  pip install pandas scikit-learn



2\. Run the project:

&#x20;  python main.py



\## 💡 Example

Enter subjects: Maths, Physics, CS  

Output: Recommended study order based on priority
